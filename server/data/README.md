# Données du jeu

Les questions et catégories sont stockées dans **MySQL**, pas dans ce dossier.

- Configure `server/.env` (voir `server/.env.example`)
- Démarre MySQL (MAMP, XAMPP, Docker, etc.)
- Lance `npm run dev` — le seed s’exécute automatiquement

**Voir les tables :** phpMyAdmin, MySQL Workbench, DBeaver, ou `npm run db:status`

Le fichier `game.db` (ancien SQLite) n’est plus utilisé — tu peux le supprimer.
